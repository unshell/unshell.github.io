词类

虽然句子才是关键，但是句子是由单词构成的，所以了解不同单词的词性也就变的极为重要

实词 Content Words

  • 能够单独充当句子成分
  • 有实在的词汇意义和语法意义
实词 英文名称 例子 英文
名词 Noun 大象 elephant
代词 Pronoun 你 you
动词 Verb 学习 study
形容词 Adjective 好玩的 interesting
副词 Adverb 高效的 efficiently
数词 Numeral 七 seven

虚词 Function Words

  • 不能够单独充当句子成分
  • 没有完整的词汇意义
  • 有语法意义
词类 英文名称 例子
冠词 Article a/an 一个，the 这个
介词 Preposition on 在….上面
连词 Conjunction if 如果
助动词 Auxiliary can 可能
感叹词 Interjection ah 啊

非谓语动词

动名词、分词和不定式被称为非谓语动词，在句子中可以充当名词、形容词和副词来使用，但唯独不能充当动词使用。

非谓语动词更像是从句的结构

句子成分

简单来说词充当句子中的成分，理解成分就能剖析句子。

主干成分

主谓宾补是句子的主干成分，所以我们很快得出了以下三个句型：

代号 句型
S + V 主语 + 谓语
S + V + O 主语 + 谓语 + 宾语
S + V + C 主语 + 谓语 + 补语/表语

然而有时候即使存在宾语和补语也不一定能把话说清楚，所以还有以下两种句型：

代号 句型
S + V + O + O 主语 + 谓语 + 宾语 + 宾语
S + V + O + C 主语 + 谓语 + 宾语 + 补语
